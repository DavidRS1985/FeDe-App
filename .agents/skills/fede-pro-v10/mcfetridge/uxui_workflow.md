# McFetridge UX/UI Workflow

A 6-phase process for implementing the artistic aesthetic into functional web interfaces.

## 01 — Research & Inspiration
- **Deliverables**: Artist reference board, Visual DNA breakdown.
- **Goal**: Extract core visual rules from art history.

## 02 — Design Principles
- **Deliverables**: Flat + Spare + Line definitions.
- **Anti-patterns**: Glows, blurs, gradients, large rounded corners.

## 03 — Token System
- **Deliverables**: CSS variables (:root), Spacing scale (base 4px), Type scale.

## 04 — Component Library
- **Deliverables**: Buttons, Badges, Form Inputs, Cards.
- **Rule**: Purely functional geometry.

## 05 — Illustration System
- **Deliverables**: SVG motifs based on "Circle + Rectangle" formula.
- **Pattern**: Composition as a puzzle.

## 06 — Implementation & Handoff
- **Deliverables**: Screen application, Consistency audit, Checklist.
